# Dataset and Storage Notes

This portfolio version has been reduced to the core files needed to understand the project.

## Included

- the main notebook
- the written report
- small derived frequent-itemset outputs

## Excluded from the public-facing version

The original working folder contained several large files that are not ideal for a normal GitHub repository, including:

- raw retail data exports
- cleaned but still heavy CSV derivatives
- item and user similarity matrices stored as `.pkl`
- compressed duplicate archives
- alternate notebook versions and notebook checkpoints

## Why they were excluded

- to stay within practical GitHub size expectations
- to keep the repository focused on the most reviewable academic deliverables
- to avoid publishing redundant intermediate artifacts
- to make the project easier for recruiters or reviewers to navigate

If needed later, a fuller archival version can be kept privately outside the public GitHub repository.
